## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

[TNW](https://github.com/javen205/TNW) example for Nest


## Installation

```bash
$ git clone https://github.com/javen205/TNW
$ cd TNW/nestDemo
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
